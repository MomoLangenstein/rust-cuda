Safety

This function is only safe to call iff cuda_repr_mut is the DeviceBoxMut borrowed on the CPU using the corresponding LendToCuda::lend_to_cuda_mut. Furthermore, since different GPU threads can access heap storage mutably inside the safe inner scope, there must not be any aliasing between concurrently running threads.
